# UI Guide

This document provides a guide to the UI of the SellerProof application.

## Colors

*   **Primary Color:** `#2196F3` (Blue)
*   **Accent Color:** `#FFC107` (Amber)

## Typography

*   **Font Family:** Roboto
*   **Font Size:** 16px

## Layout

*   **Spacing:** 8px
*   **Padding:** 16px
*   **Margin:** 16px

## Components

*   **Buttons:** Use `ElevatedButton` for primary actions and `TextButton` for secondary actions.
*   **Text Fields:** Use `TextField` for text input.
*   **Cards:** Use `Card` to display information in a container.
*   **Lists:** Use `ListView` to display lists of items.
*   **Dialogs:** Use `AlertDialog` to display dialogs.
*   **Snackbars:** Use `SnackBar` to display snackbars.
*   **Bottom Sheets:** Use `showModalBottomSheet` to display bottom sheets.
*   **Navigation:** Use `Navigator` to navigate between screens.
